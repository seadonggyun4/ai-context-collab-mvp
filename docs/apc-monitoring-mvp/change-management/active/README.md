# Active Change Requests

`REQUESTED`부터 `SELF_REVIEWED`까지 진행 중인 Change Manifest를 둔다.

- 새 요청은 `../templates/change-manifest-template.md`를 복사해 시작한다.
- 파일명은 `CR-YYYY-NNN_short-title.md` 형식을 사용한다.
- 하나의 요청은 하나의 활성 Manifest만 가진다.
- `COMPLETED` 또는 `CANCELLED` 전환 후에는 `../archive/YYYY/`로 이동한다.
- 활성 요청이 없다면 이 README만 남긴다.
