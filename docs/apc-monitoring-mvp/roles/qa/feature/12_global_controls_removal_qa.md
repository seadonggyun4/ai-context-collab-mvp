## QA 목적

실제 동작이 없거나 탭과 중복되는 전역 UI가 제거되고, 각 기능 탭 내부의 필요한 기능은 유지되는지 검증한다.

## 체크표

| 체크 항목 | 확인 |
| --- | --- |
| 기준일/APC/품목/입고·선별/상태 전역 필터 panel이 제거되었는가 | [ ] |
| 헤더 우측 현재 역할 selector가 제거되었는가 | [ ] |
| 헤더 우측 역할 chip이 제거되었는가 | [ ] |
| 헤더 우측 품질 이슈 버튼이 제거되었는가 | [ ] |
| 헤더 우측 Excel 다운로드 버튼이 제거되었는가 | [ ] |
| APC 데이터 관리 탭은 유지되는가 | [ ] |
| 수신 현황 내부 수신 조건 selectbox는 유지되는가 | [ ] |
| 데이터 조회 탭 내부 Excel 다운로드 흐름은 유지되는가 | [ ] |
| 운영 조치 등 기능 화면 내부 권한 표시는 유지되는가 | [ ] |
