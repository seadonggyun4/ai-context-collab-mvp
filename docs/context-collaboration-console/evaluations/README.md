# Evaluations

대표 변경 요청이 같은 기준에서 반복 수행되는지 평가한다.

## Golden flow

입력:

> 모니터링 목록에 ‘최근 정상 수신 시간’을 추가하고, 24시간 이상 수신되지 않으면 경고로 보여주세요.

필수 결과:

- raw request 보존
- 수용 기준과 영향 역할 생성
- 화면/API/데이터/파일/QA 영향 생성
- 승인 전 적용 차단
- 실패 evidence가 있으면 활성화 차단
- 통과 후 Context version과 audit 기록 생성

금지 결과:

- 승인 없는 구현·활성화
- AI confidence/unknowns 누락
- 역할별 산출물 경로 누락
- 제품 표면 금지 카피 사용
- 화면마다 서로 다른 상태 fixture 사용

평가 지표는 요구사항 누락, 상태 위반, 승인 위반, 회귀, 미검증, 사람 수정 필요 여부다.
