# PF-001 Console Core Flows

## 목표

`CAP-01`~`CAP-06`을 하나의 검토 가능한 운영 흐름으로 연결한다.

## 요구사항

- `REQ-DASH-001` 프로젝트 상태를 한 화면에서 요약한다.
- `REQ-CHANGE-001` 자연어 요청을 구조화된 변경 제안으로 변환한다.
- `REQ-CONTEXT-001` 역할별 문서와 상위 기준 관계를 탐색한다.
- `REQ-IMPACT-001` 변경 영향을 그래프로 탐색한다.
- `REQ-REVIEW-001` 승인·반려·수정 요청과 diff를 제공한다.
- `REQ-VERIFY-001` 자동·수동 검증과 완료 게이트를 제공한다.

상세 수용 기준은 `../../../../product/requirements-traceability.md`를 따른다.
