# QF-001 Console Acceptance QA

## P0 시나리오

1. `apc-monitoring-mvp` 대시보드에서 미승인·미검증 요청을 찾는다.
2. 예시 자연어 요청을 제출하고 분석 결과를 확인한다.
3. 역할·화면·API·데이터·파일·QA 영향이 모두 표시되는지 확인한다.
4. 승인 전 적용이 불가능한지 확인한다.
5. 승인 후 구현·검증 상태로 이동하는지 확인한다.
6. 실패 테스트가 있으면 완료와 Context 활성화가 차단되는지 확인한다.
7. 통과 후 새 Context version과 감사 이력이 표시되는지 확인한다.

상세 케이스는 `../../../../qa/test-matrix.md`를 따른다.
